# Ecommerce Platform – Java Application

## 📌 Project Overview

This project is a simple Ecommerce Platform developed using Java. The system allows users to browse products, add items to the cart, and manage orders.

## 🎯 Objective

The main objective of this project is to understand the working of an online shopping system and implement core ecommerce functionalities using Java.

## 🚀 Features

* Product listing
* Add products to cart
* Order management
* Basic user interaction system

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Console-based interface

## ⚙️ How the System Works

1. The user views available products.
2. The user selects products to add to the cart.
3. The system manages selected items.
4. The user can place an order.
